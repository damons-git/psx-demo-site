Bluetooth PSX Controller

Description

This page documents how I transformed an outdated and redundant original Sony Playstation (PSX) DualShock controller into a modern bluetooth variant.

The motivation behind this build stemmed from seeing a few attempts from other DIY modders, such as reddit user u/Dementat_Deus, and DIY blogger and creator Fabio Angelitti.

But without further ado, my pièce de résistance!

Components List

  • Sony Playstation DualShock controller (model SCPH-1200)
  • 250mAh Lithium Polymer (LiPo) Battery
  • TP4056 Lithium Charger Module
  • 3.5mm x 1.3mm PCB DC Power Barrel Jack (Female)
  • 3.5mm x 1.3mm Barrel Jack Lead (Male)
  • Mini Voltage Step-up module
  • Arduino Pro Mini 5v 16MHz
  • Singular Neopixel
  • HC-05 Bluetooth Module
  • AMS1117 3.3v Step-down Voltage Regulator
  • Vero Stripboard (Used for fabrication of mounts)
